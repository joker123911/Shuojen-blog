這篇文章記錄我對抗 1.e4 的核心武器：法蘭西防禦。我依據對手的不同回應，拆解成以下幾個主要章節。

## 第一章：交換變化 (Exchange Variation)
當白棋選擇 3.exd5，局面會變得極度對稱。我們的目標是在穩固中尋找反擊，特別要注意白棋孤兵（Isolated d-pawn）的弱點。

<ChessStudyBlack pgn="1. e4 e6 2. d4 d5 3. exd5 exd5 4. Nf3 Nf6 {此處一定要注意出動王翼象} (4... Bd6 5. Bd3 Ne7)" />

---

## 第二章：進兵變化 (Advance Variation)
當白棋走 3.e5 試圖封鎖中心。這是法蘭西防禦最經典的戰場！我們的核心戰略是**瘋狂進攻白棋的 d4 兵鏈基地**。

<ChessStudyBlack pgn="1. e4 e6 2. d4 d5 3. e5 c5! {立刻反擊中心} 4. c3 Nc6 5. Nf3 Qb6 {形成標準的法蘭西施壓陣型}" />

---

## 第三章：塔拉什變化 (Tarrasch Variation)
白棋走 3.Nd2 是為了避免象被牽制，這條線比較考驗內功。

<ChessStudyBlack pgn="1. e4 e6 2. d4 d5 3. Nd2 c5! {立刻敲開中心} 4. exd5 Qxd5" />

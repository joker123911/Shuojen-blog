# 機械鍵盤
最近朋友在群組問我有沒有推薦的無線紅軸鍵盤，於是興起念頭想寫一篇機械鍵盤的文章。

## 入坑故事
最開始入坑機械鍵盤是在 2019 的時候，當時主宰機械鍵盤市場的是 Filco 系列的鍵盤，櫻桃（Cherry）機械軸的茶青黑紅軸是帝王級的存在，手感的代名詞，於是我入手了 Filco 的忍者茶軸，俗稱「忍茶」。

![img](../blog/img202512/041.webp)

換鍵帽是機械鍵盤非常好玩的一個特色，雖然忍者系列的側刻品質很好，也很漂亮，但是由於 ABS 的材質較容易有打油的情況，捨不得多用，於是我入手了一副台灣製造的藍粉太豪鍵帽，非常亮眼，到現在還在我的 Leobog Hi98 上服役中。

![img](../blog/img202512/042.webp)

## 中國鍵盤崛起
時間一轉眼到了 2023 年，好朋友大隊長神秘兮兮的把我跟王胖找去他家，展示了他在淘寶上研究中國品牌的鍵盤，我才知道櫻桃軸與傳統幾個鍵盤大廠的硬體已經完全跟不上中國的產品，從這時候我也入坑開始研究中國鍵盤市場和軸體。

大隊長的軸體火力展示，各種琳瑯滿目的軸體。印象最深刻的是凱華的重力青軸，跟櫻桃青軸比起來，真的天差地遠，集中沉穩又Q彈，我馬上買了一套（結果都沒在用），大隊長服役的是手感很棒的冰淇淋軸，CP 值最好的是灰木軸，靜音的我最喜歡青檸軸（收了一套之前在辦公室用）。

![img](../blog/img202512/052.webp)

![img](../blog/img202512/051.webp)

從 2014 年櫻桃 MX 軸體專利到期後，中國國產軸體（凱華、TTC、佳達隆）的技術突飛猛進，手感與聲音都頂天，價格又漂亮。中國鍵盤品牌（賤驢、RK、VGN、Leobog）開始將原本昂貴的客製化功能—Gasket結構、全鍵熱插拔、PBT鍵帽、三模連接（Type-C、藍芽、2.4）下放至千元台幣價位帶 ， Filco、Leopold 及其他電競品牌的傳統大廠，還是像過去一樣過度依賴品牌溢價，已經不再受市場青睞了。

## 我的鍵盤

當時最開始我組了一把賤驢的 RS2，配上有注音的啟示鍵帽還有 CP 值非常高的無段落灰木軸，現在在公司的辦公桌上服役中。

![img](../blog/img202512/047.webp)

![img](../blog/img202512/050.webp)

RK家的 R75，其實買來不知道要幹嘛，但是就是太便宜了，大約一千初，隨便買手感都好，後來送給朋友。

![img](../blog/img202512/053.webp)

大約 2024 年，CNC全鋁外殼的「鋁坨坨」鍵盤開始從高價下放，鋁製外殼的紮實度十足，拿起來很沉，打字的聲音更結實集中，會有所謂的「麻將音」和「石子音」，不需要做什麼麻煩的調整就很好聽，非常過癮。

我最愛的鍵盤 Leobog Hi75，漢白玉軸配上全白鍵帽，開箱即用，手感聲音都太美妙了。

![img](../blog/img202512/056.webp)

購入 Leobog Hi98 在房間內使用，我覺得打起來手感還是 Hi75 聲音最好，不過 98 排列有數字鍵，寫部落格文章比較好用，白色幫其他朋友買的，我買黑色的。

![img](../blog/img202512/059.webp)

將 Hi75 換一套有注音和倉頡的鍵帽，放在客廳的 linux 電腦使用，

這時候的我已經將 Filco 塵封了，雖然直接打也不會覺得有什麼問題，但是沒有比較沒有傷害，現在已經回不去傳統大廠了。

## 磁軸鍵盤

後來時不時我跟大隊長、王胖都會聚在一起集運鍵盤，在遊戲市場（特別是 FPS 射擊遊戲），還有很重要的「磁軸鍵盤」。它的核心技術並非傳統的「金屬接觸導電」，而是透過感應軸體內部磁鐵的距離變化來觸發訊號。磁軸鍵盤能做到***只要手一抬起，訊號立刻中斷；只要手一按下，訊號立刻觸發***，在 FPS 遊戲中，磁軸能讓你比對手更快停下準心開槍，算是一種「物理外掛」，不過已經電子陽痿的我，就沒跟著買了。

## 結論

現在 2025 這個時候，我個人會推薦去淘寶買鍵盤，不只品質比較好，價格也完勝，可以在兩千台幣內買到三模、可熱插拔更換軸體、聲音又非常好的鍵盤。 傳統大品牌的 Cherry 軸已經被遠超了，不如凱華軸跟 Leobog 的軸體，買淘寶的話，建議當成沒保固，有風險還是要自己評估。

配列需求分為以下幾種：

* 104 鍵全尺寸，最完整但是需要比較大空間。
* 100/98 鍵少一些不常用的鍵，但還是有數字區可以用，是現今最主流的配列。 
* 87/75 鍵無數字鍵區。
* 60 鍵更小更緊湊，鍵盤會很簡潔漂亮但使用上比較不方便。

淘寶品牌推薦 Leobog， Hi75 手感音色都超級好，要靜音我推薦很安靜的青檸軸，在辦公室打很好用，要無段落的手感就選 CP 質高的灰木軸。 差不多一千多台幣本體，加上軸跟喜歡的鍵帽各三百台幣，兩千台幣內搞定。

如果怕麻煩也沒追求 CP 值，或單純不想買淘寶，可以買 Filco、Leopold
但都要三四千台幣起跳，我覺得不值得這個價，低一點就選 Ducky 或 Irock，可能二到三千台幣可以搞定。詳細型號我就沒辦法推薦了，這幾個品牌都很穩，不會有什麼錯，就是 CP 值低了點，換保固比較安心，適合沒追求的大眾。

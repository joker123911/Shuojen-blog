最近晚上跟蝦波一起看[《殺手青春》](https://zh.wikipedia.org/zh-tw/%E6%AE%BA%E6%89%8B%E9%9D%92%E6%98%A5)，每次看到主角吃拉麵的片段，就會好想吃拉麵！今天來整理一下我吃過的幾間拉麵店，我的評分標準就是好吃、價錢合理、最好不用排隊。

## SSS Tier

| 店名 |短評 | 
| --- | --- | 
| 麵屋雞金 | 我吃過的拉麵店中最好的麵條（我都選最硬），完美的扒住濃濃的雞白湯頭，加一堆蒜泥超讚。離家近還不用排隊，營業到很晚隨時想吃都可以，老闆人很好，實在是完美的一間店。 | 
| 你回來了 | 店面改版前後都非常美味，味道絕對可以彌補需要排隊的缺點，早營業時間 10 分鐘到就可以不用等很久了，叉燒湯頭麵條各方面都很頂尖，加點一份高麗菜也很讚。 | 
| 柑橘Shinn | 多年前吃到一次就很愛，清新淡雅的風味，可是又不會覺得淡，快吃完時留個三成份量，再加一次高湯，會有兩種不同的享受！但這間要排挺久，所以越來越少去了。 | 
| 麒麟創作拉麵坊 | 抽號碼牌後還逛了一個多小時的街才吃到，所以人生只吃過一次。但是真的滿驚豔的，很美味，加麵的時候更驚喜了，竟然還有肉塊，小確幸真的必須讚。 | 
| 麵屋壹の穴 | 離家近所以常吃，改版前後都不錯，以前有 99 元的無印版醬油拉麵超讚，可惜之後都沒看到了，雞白湯、煮干系、二郎系都有，我喜歡雞白湯 = 二郎系 > 煮干系。 | 

## SS Tier

| 店名 |短評 | 
| --- | --- | 
| 雞二拉麵 | 雞白湯版本的二郎系，可以點份量超多也有一般份量，可惜要排隊，如果不用的話我會將這間升級到 SSS Tier。 | 
| 拉麵魚堺 | 表現很好的一間清淡系拉麵，醬油或鹽味都不錯，但也是要排隊，就不會想特別跑一趟去吃了。 | 
| 豚人拉麵 | 好吃，有很酷的天龍麵，特別限定的二郎系也很讚，吃起來很過癮，人不會很多。 | 
| 赤坂拉麵 | 已經消失的一間店，小學時的美好記憶，算是我對日式拉麵的啟蒙。 | 
| 福岡屋拉麵 | 從小吃到大，平實規矩的一間店，風格改良的較台式，但也很好吃，白湯很讚。 |

## S Tier

| 店名 |短評 | 
| --- | --- |
| 森本家拉麵 | 從國中跟大隊長吃到現在，平實的好滋味，不太有記憶點但是有很多朋友的回憶。 | 
| 真劍拉麵 | 有很神奇的藍色拉麵，控肉也很特別，但是要排隊就沒有特別想過來吃的慾望。 | 
| 數寄屋 辻葉 | 好吃，湯頭很濃郁香醇，有經過的話會想吃，但也是要小排隊的一間店。 |
| 豚將拉麵 | 大學時看朋友加五球麵的一個好地方，平實 CP 值就是這間店的意義。 | 
| 一幻拉麵 | 在東京吃的，印象中是很濃郁的蝦頭鮮味，還不錯，不過在臺灣沒吃過。 |
| 墨洋拉麵 | 干貝表現很好，招牌海味滿特別的，但是價格有點高又要排隊，比較沒有再訪的念頭。 |
| 一蘭拉麵 | 其實是好吃的一碗麵，不會對味道有什麼意見，但是 CP 值很低，優點是半夜可以吃吧。 |
| 麵堂拉麵 | 在新竹的時候很常吃，離宿舍很近，感謝把我餵飽的一間店，牛肉很軟嫩又香，推推。 |
| 麵屋浩 | 新竹有名的一間，料很多很濃，老闆人好，很讚，但是要排隊，可惜。 |
